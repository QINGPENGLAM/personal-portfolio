    Game for Art history

    Reflection

    What materials did you use

    Tool I used Unity and research I used youtube, khan academy, internet (link at the end)

    Briefly describe the overall process and steps.

    Overall process is to determine what topic to choose from it will determine the scene you're creating for example if you're doing artifact you will need the original location where it was found if you choose a artist you will need a scene of that particular time period.

    After choose my topic of art artifacts I start working on user interaction system since Im doing a game design where user were able to interact I need to set the camera, the lighting, the texture, the model, the code.

    Most of the time I'm just doing research when I run into trouble but when I finish up all the coding end I start doing artifact research the detail about the artifact, the picture, the origin, the material.

    Tips for other who want to do similar CS project is plan out what needs to be done because you might have all the idea all at once but when it's all together you get confused easily. Google is your best friend.

    Purpose + Goal

    Why did you pick this topic/ project?

    During the time I was learning computer programming and a mini game called oregon trail it's a 2D adventure game at beginning I was trying to apply the setting into this project and later I found that its not that easy and when I search around on youtube I saw unity and I unity is a efficient tool to create a game like project so I started my project.

    Theme I want to express is that current society we dont have time and resources to experience what happen to history and as technology becomes more advanced maybe we can design a VR that teaches art history but in a fun and engaging way

    I try to be as specific as possible and be visual than wording.

Also at the beginning I waste a lot of time trying to only used coding to present image and user interaction I should just start the unity part.
